# raindrop-mcp

MCP for raindrop.io bookmark service in Node.js

## Setup

```bash
pnpm install
```

## Build

```bash
pnpm build
```

Compiles TypeScript to JavaScript in the `dist/` directory.

## Development

```bash
pnpm dev
```

Runs the TypeScript compiler in watch mode for development.

## Run

After building:

```bash
node dist/index.js
```

## Clean

Remove build artifacts:

```bash
pnpm clean
```
